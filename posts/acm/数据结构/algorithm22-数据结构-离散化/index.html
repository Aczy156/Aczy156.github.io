数据结构 离散化

实质上是一种哈希，将原数组进行哈希映射，映射前后仍然保持全序/偏序

基础实现

数据结构：构造结构体，重载运算符（val和idx一次排序）。

1、根据val来排序，如果val相同的话，就根据idx索引递增的次序

2、再将每个位置反向赋值回去。

struct DisNum {
    int idx, val;
    bool operator < (const DisNum n) const {
        if (val == n.val) return idx < n.idx;
        return val < n.val;
    }
}a[maxn];
